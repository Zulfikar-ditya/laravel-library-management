# Get Started
How To Use This App

## Step 1
> run composer command
```
composer install
```

## Step 2
> run npm if needed
```
npm install
```

## Step 3
> configure database, make a database with name:
```
library_management
```

## Step 4
> make a copy and edit file .evn
run this
```
cp .env.example .env
```
> change database name. and configure the others

## Step 4
#### Migrate The Database
> run this command on this folder
```
php artisan migrate
```
## Step 6
> run server
```
php artisan sever
```
